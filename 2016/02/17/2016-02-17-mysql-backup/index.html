阅读量:

在Java项目中，可能会定时对数据库进行备份。

可以使用Mysql自带的mysqldump命令，具体命令为：

在备份时通常备份一个数据库

所以可以使用命令如下：

-h 主机地址
-P MySQL端口
-u 用户名
-p 密码
–default-character-set 字符集
-B 数据库
-r 导出文件
注意：在-p后紧跟参数 在-B后必须空格 其它都可以
